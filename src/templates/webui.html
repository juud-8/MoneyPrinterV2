Start in 30 seconds

          
    
            
  1. From the project root in PowerShell: $env:PYTHONIOENCODING = "utf-8"
    2. 
            
  2. Run .\venv\Scripts\python.exe src\webui.py
    3. 
            
  3. Open http://127.0.0.1:5757 (this page)
    4. 
          

        

        

          
Daily loop

          
    
            
  1. Brands → Generate only (review output under output/<brand>/)
    2. 
            
  2. Pipeline → watch the filing theater (or See terminal)
    3. 
            
  3. When ready: Generate & Post now
    4. 
            
  4. Click Refresh YouTube metrics after uploads land
    5. 
          

        

        

          
Weekly review

          
    
            
  1. Refresh metrics (or rely on Sunday Task Scheduler job)
    2. 
            
  2. Check Overview winners on the leaderboard + brand insights
    3. 
            
  3. Check Spend tab vs your alert threshold
    4. 
            
  4. Optionally set Studio retention % via CLI (see Performance tab)
    5.
