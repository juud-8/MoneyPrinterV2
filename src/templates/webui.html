Daily loop

          
    
            
  1. Overview → Attention: clear any anomalies first (silent upload failures, stale metrics)
    2. 
            
  2. Brands → Generate only, review the render under output/<brand>/ (Open folder button)
    3. 
            
  3. Pipeline → watch the filing theater (or See terminal)
    4. 
            
  4. When ready: Generate & Post now
    5. 
            
  5. Hit ⟳ Metrics after uploads land
    6. 
          

        

        

          
Systems LEDs

          
    
            
  • green — service reachable / key present / path valid
    • 
            
  • amber — degraded: stale metrics, low disk, optional key missing
    • 
            
  • red — down: Ollama unreachable, missing profile, no config
    • 
            
  • Click the strip to re-run the check. Full preflight: python scripts/preflight_local.py
    • 
          

        

        

          
Blank numbers?

          
    
            
  • Views/likes need youtube_api_key in config.json + Refresh metrics
    • 
            
  • Growth Terrain needs at least one metrics refresh (channel snapshots)
    • 
            
  • Topic insights activate at 5+ uploaded Shorts older than 48h with views
    • 
            
  • CTR / RPM / affiliate clicks are not auto-filled (Studio-only)
